How to create a store?

In this example we going to go through the steps of creating an Ebay store
To do this there are two things that we need:

The first is the tag name and the second is the store query.

The tag name and story query enables the app to query the price for any item that belongs to this store



First, go to the website of your choice. In this example we are using the Ebay website. This will be our store for the app.

Select any item from the website. The item selected must show the full price of the item and not the price range.

For example, the price must not be a range of two numbers, e.g £20 - £30 but a single confirmed price e.g. £25 .

In this example we using the item WX255L WORX SD SemiAutomatic Driver w/ Screw Holder and the price for the item is $21.99

Highlight the price of the item and right click the mouse. A popup box should appear. Select the option inspect element. You should see something similar to the highlighted picture below.


Now, right click the mouse on the highlighted link show above to open the dialogue box show below




The first element in the bracket is usually the tag and it can be li, span, div, etc.

In this case the tag name is span. Now we need to select the query. The query is the id="some value", class="some value" or the the itemprop="some value". Note these elements might vary.

Select one or all of them. Make a mental note or write the one you have selected. In this case we are going to select the itemprop="price"
Now we are ready to create our store.





To create a store click on my store locate in the above picture. After clicking on the button you be giving the options to create a new store. This would look like the picture below

Click the create new store button


You should now be greeted by a row of tables. The tag name and query should already be filled in. For now leave that we get to it in a bit.


Fill in the store details e.g the title, description and the url prefix. For the tag name enter the tag name you had taken earlier. Replace the default query with elements of the new query.

In our early query we made a mental note of the query itemprop=price
Now replace the default string id and price-now with the new query{"id": "price-now"}

In our case the new query will be {"itemprop": "price"}

The itemprop and price must be placed inside double quotes. Note we could entered the store query as {"itemprop": "price", "id": "prcIsum", "class": "notranslate"} and it would still would have worked


Congratulations, you now have a store
