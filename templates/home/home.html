What is Price Alerter?

Price Alerter is an online web based application that allows users to track the prices of any *online item*. The app allows the user to create a given store along with the price of the item they want to track. The app will then monitor the price of the item and if the prices changes the user would be alerted of the new price by email. The site comes with three predefined stores Ebay, Amazon and Game that allows the user create an item which the user can then add an alert to.

Why would I want to use price alerter?

Suppose you go to an online website and you see an item you really want to buy. But unfortunately the price for the item is way too high. And this point you left with two choices buy the item or wait for the price to down.

Now suppose you chose the latter, you then left with another choice on how to check the item price. One way is to save the item's web page to your bookmark and then check the prices hourly or daily. When the price drops you can then purchase the item. Great, but it seems like a bit hassle just to buy an item. What if you want buy multiple items?

The second way is to use Price Alerter which does the the same job but without you lifting a finger. Well, you might have to lift a few fingers, create the store, create an item to monitor as well as the price, add the url but that it, I swear.

Great, but how does price alerter work?

Price Alerter works by allowing the user to set a price alert for a specific product/item from any store listed in the website e.g Ebay, John Lewis, etc, the user wants to buy. The user can also create a new store if the store is not listed in the list of the pre-defined stores.

The application then monitors the price behind the scenes regulary checking if the price has dropped and when the item's price dropped an email is sent to the user informing of the price drop. Now the user is free to purchase the item saving them a ton of money or multiple items.
